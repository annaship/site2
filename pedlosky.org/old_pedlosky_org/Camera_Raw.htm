Spring Semester 2008

4.              CameraRaw Workshop

taught by Holly Smith Pedlosky

For intermediate photographers with cameras which can shoot Raw files

one day, Saturday 2/16/08, 10am-4:00pm.

In this digital darkroom workshop you will learn how to convert Camera Raw files into dynamic photographs with Adobe Photoshop CS3. At the end of the day, you will be able to make decisions about exposure, hue, contrast, resolution, bit-depth, and color balance with Camera Raw. You will be able to make reversible, non-destructive changes using Smart Layers.

Adobe Photoshop CS3’s powerful new Raw Conversion engine allows you to use a digital zone system and make exciting, high dynamic range photographs that would make Ansel Adams dance with joy.

For intermediate photographers with cameras which can shoot Raw files; one day, Saturday 2/16/08, 10am-4:00pm. Room: Library 401.
